# Canoe Routes Mapper

Early days of a route mapping tool for Canoe Routes Ontario.

## Map Library

This app uses [Leaflet JS](https://leafletjs.com/), an open-source mapping library.

## Local UI

To build and serve the app locally, navigate to the route and run:

```bash
npm i
npm start
```

Once ready, the app will be available at `http://localhost:3000`
