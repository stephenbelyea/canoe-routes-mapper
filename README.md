# Canoe Routes Mapper

Early days of a route mapping tool for Canoe Routes Ontario.

## Local UI

From the repo's root, run:

```bash
npm install
npm run deps
npm start
```

Once built, the page will be served at: `http://localhost:3000`

## GPKG Reader Script

From the repo's root, run:

```bash
npm install
npm run parse
```

After the parser runs, check the JSON files within `scripts/output` for updated content.
